Цели и задачи

  • Познакомиться с операционной системой Linux. Получить практические навыки рабо- ты с редактором vi, установленным по умолчанию практически во всех дистрибутивах.

Выполнение лабораторной работы

  1. Создаю каталог с именем ~/work/os/lab06 и перехожу в него

  1. Вызываю vi и создаю файл hello.sh

  1. Нажимаю клавишу “i” и ввожу следующий текст #!/bin/bash HELL=Hello function hello { LOCAL HELLO=World echo $HELLO } echo $HELLO hello ; нажимаю клавишу “Esc” для перехода в командный режим после завершения ввода текста.

  1. Нажимаю “:” для перехода в режим последней строки; нажимаю w(записать) и q(выйти), а затем нажимаю клавишу “Enter” для сохранения моего текста и завершения работы

  1. Делаю файл исполняемым

  1. Вызываю vi на редактирование файла

  1. Устанавливаю курсор в конце слова HELL второй строки, перехожу в режим вставки и заменяю на HELLO ; нажимаю “Esc” для возврата в команднйы режим

  1. Уставливаю курсор на четвертую строку и стираю слово LOCAL ; перехожу в режим вставки и набераю текст : local ; нажимаю “Esc” для возварта в командный режим

  1. Установливаю курсор на последней строке файла. Вставляю после неё строку, содержащую следующий текст: echo $HELLO.

  1. Нажимаю “Esc” для перехода в командный режим. Удалите последнюю строку.

  1. Ввожу команду отмены изменений “u” для отмены последней команды. Введите символ “:” для перехода в режим последней строки. Запишите произведённые изменения и выйдите из vi.
